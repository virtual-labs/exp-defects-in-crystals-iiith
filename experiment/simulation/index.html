Instructions
  • The experiment aims to visualize and verify the reciprocal lattice for a given lattice
  • 'Select Atoms' button can be used to select the atoms that represent the reciprocal lattice for the given lattice
  • 'select Region' button can be used to visualize the convex polygon which encapsulates the selected atoms.
  • 'Check reciprocal lattice ' button can be used to verify if the chosen points form the reciprocal lattice or not.
  • The results will be displayed in the observations section on the right.
